# Installation
## Dependencies
Make sure to have sdl2 installed on your system
- Ubuntu: `sudo apt install libsdl2-dev`
- Arch Linux: `sudo pacman -Sy sdl2`

<br />

## Pull and Build
- `git clone https://github.com/OliverBenz/Game_Jetgame_New`
- `cd Game_Jetgame_New`
- `mkdir build; cd build`
- `cmake ..`
- `make`
- `./src/main`
