# Jetgame
C++ Refactored Version of the 2016 Slick2D JetGame

## Controls

### General
`R` Restart game

<br />

### Left Player
`W` move up <br />
`A` move left <br />
`S` move down <br />
`D` move right <br />
`Space` shoot

<br />

### Right Player
`Arrow Up` move up <br />
`Arrow Left` move left <br />
`Arrow Down` move down <br />
`Arrow Right` move right <br />
`Numpad 0` shoot
